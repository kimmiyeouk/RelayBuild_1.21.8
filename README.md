# RelayBuild Plugin

**릴레이 건축(Relay Build) 플러그인**은 유튜브 쇼츠 또는 라이브 콘텐츠 제작을 위해 기획된 마인크래프트(Minecraft) Spigot 서버용 플러그인입니다. 여러 플레이어가 정해진 시간 동안 인벤토리를 승계받으며 하나의 건축물을 이어 만들어나가는 릴레이 게임을 진행할 수 있습니다.

## 개발 환경

* **플러그인 버전:** 1.0
* **지원 마인크래프트 버전:** **1.21.x** (1.21.8 이상 권장)
* **지원 버킷:** **Spigot**
* **필요 자바(Java) 버전:** **JDK 21**

## 게임 규칙

1.  **주제 설정:** `/relay start <주제>` 명령어로 게임을 시작하며 주제가 설정됩니다.
2.  **준비 시간:** 1회차 첫 번째 플레이어에게만 10초의 아이템 준비 시간이 주어지며, 이 시간 동안은 건축/파괴가 불가능합니다.
3.  **건축 시간:** 준비 시간이 끝난 후, `config.yml`에서 설정한 **N초** 동안 건축 및 파괴가 허용됩니다.
4.  **인벤토리 승계:** 건축 시간이 종료되면, 다음 순서 플레이어는 이전 플레이어의 인벤토리를 그대로 물려받아 건축을 이어갑니다.
5.  **라운드 반복:** 모든 참가자가 한 번씩 건축을 마치면 1라운드가 종료되며, 설정된 **M라운드**만큼 게임이 반복됩니다.

## 설정 (config.yml)

플러그인 폴더 내 `config.yml` 파일을 열어 게임 규칙을 변경할 수 있습니다.

| 변수 이름 | 기본값 | 설명 |
| :--- | :--- | :--- |
| `N` | `60` | 한 플레이어에게 주어지는 **실제 건축 시간** (초) |
| `M` | `2` | **총 진행할 라운드(회차) 수** (예: 2로 설정 시 두 번씩 건축) |
| `PlayerTag` | `"builder"` | 게임 참가자를 식별하는 태그. 이 태그를 가진 플레이어만 참가합니다. |

## 명령어 (Commands)

모든 명령어는 OP 권한 또는 적절한 권한을 가진 플레이어만 사용할 수 있습니다.

| 명령어 | 설명 |
| :--- | :--- |
| `/relay setA` | 현재 위치를 **건축방 텔레포트(A 지점)** 위치로 설정합니다. |
| `/relay setB` | 현재 위치를 **대기방 텔레포트(B 지점)** 위치로 설정합니다. |
| `/relay start <주제>` | **게임이 시작됩니다.** 명령 실행 전 반드시 A, B 지점 설정과 참가자 태그 부여가 완료되어야 합니다. |

### 게임 시작 전 필수 준비 사항

1.  **참가자 태그 부여:** 게임에 참여할 플레이어에게 `config.yml`에 설정된 태그를 부여합니다.
    * 예: `/tag add [플레이어 닉네임] builder`
2.  **위치 설정:** `/relay setA` 와 `/relay setB` 명령어를 통해 건축이 진행될 장소와 대기할 장소를 설정합니다.
